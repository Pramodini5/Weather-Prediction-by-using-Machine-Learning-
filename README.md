# Weather Prediction using Machine Learning

## Overview
This project predicts weather conditions using Machine Learning techniques based on historical weather data.

## Problem Statement
Accurate weather prediction helps in agriculture, disaster management, and daily planning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Machine Learning Workflow
1. Data preprocessing
2. Feature selection
3. Model training
4. Prediction
5. Evaluation

## Model Used
- Logistic Regression / Random Forest / Decision Tree (update with your model)

## Results
The model predicts weather conditions with good accuracy.

## How to Run
1. Download dataset
2. Open Jupyter Notebook
3. Run all cells
